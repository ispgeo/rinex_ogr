Rinex

(GDAL/OGR >= 2.2.3)

This driver currently read rinex 2.* files and still in development.

Available fields in features:

  • default geometry - point 3D, approx position;
  • Position = point 3D, position by *N GPS file;
  • MarkerName - string;
  • AntennaModel, AntennaType - string;
  • RecieverModel, RecieverType, RecieverVersion - string;
  • FirstObsDateTime - date time;
  • FirstObsTimeType - string (GPS, GLO, GAL);
  • LastObsDateTime - date time;
  • LastObsTimeType - string (GPS, GLO, GAL).
